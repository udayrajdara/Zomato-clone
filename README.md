# Zomato Clone

A modern and responsive Zomato-inspired web application built with React.js. The project allows users to explore restaurants, view food listings, and submit contact inquiries through EmailJS while consuming dynamic data from a Mock API.

## 🚀 Live Demo

https://zomatoo-cloone.netlify.app/

## 📖 About The Project

This project is a frontend recreation of the Zomato platform, developed using React.js to showcase modern web development practices. The application fetches restaurant and food-related data from a Mock API, providing a dynamic and interactive user experience.

Additionally, EmailJS integration enables users to submit contact forms without requiring a backend server.

The project demonstrates skills in React development, API integration, responsive design, and third-party service integration.
